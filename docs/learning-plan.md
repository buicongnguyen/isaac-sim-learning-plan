# Isaac Sim Learning Plan

Use this file as the editable tracker. Change `[ ]` to `[x]` as each item is completed, and add notes under each phase.

## Phase 1: Setup

- [ ] Confirm Isaac Sim 5.1.0 launches on this machine.
- [ ] Open a default sample scene.
- [ ] Confirm GPU rendering works.
- [ ] Confirm the Python environment can run Isaac Sim scripts.
- [ ] Save one screenshot or short recording as proof of setup.

Notes:

## Phase 2: Core Isaac Sim

- [ ] Learn stage, world, prim, asset, and USD basics.
- [ ] Add a robot asset to a scene.
- [ ] Configure physics and collision.
- [ ] Add camera output.
- [ ] Add lidar-like or depth-sensor output.
- [ ] Build a simple action graph or scripted robot movement.

Notes:

## Phase 3: ROS 2 Bridge

- [ ] Install or confirm ROS 2 environment compatibility.
- [ ] Publish camera or sensor messages from Isaac Sim.
- [ ] Subscribe to velocity or joint commands.
- [ ] Confirm TF tree or frame data.
- [ ] Drive a simulated robot using ROS 2 commands.

Notes:

## Phase 4: Reproduce Course-Style Assignments

- [ ] Review University of Miami CSC398 Assignment 2.
- [ ] Reproduce a laser-scan processing exercise.
- [ ] Review University of Miami CSC398 or CSC752 localization assignment.
- [ ] Reproduce a particle-filter localization workflow.
- [ ] Review University of Miami CSC752 Assignment 6.
- [ ] Reproduce an RRT-style path-planning workflow.

Notes:

## Phase 5: Robot Learning Bridge

- [ ] Review Cornell CS 4756/5756 FetchReach-v4 assignments.
- [ ] Map the Gymnasium/MuJoCo learning concepts to an Isaac Sim or Isaac Lab task.
- [ ] Pick one small task: reaching, navigation, or manipulation.
- [ ] Train or test one baseline policy.
- [ ] Record the gap between MuJoCo/Gymnasium and Isaac Sim workflows.

Notes:

## Phase 6: Portfolio Project

- [ ] Define one end-to-end demo.
- [ ] Include simulation scene setup.
- [ ] Include sensor output.
- [ ] Include control or planning.
- [ ] Include learning or policy evaluation if practical.
- [ ] Publish README, instructions, and a demo video.
