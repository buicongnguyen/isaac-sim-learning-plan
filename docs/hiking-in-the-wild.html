Reproduction Goal

Reproduce the core Isaac Sim and Isaac Lab ideas from Hiking in the Wild: A Scalable Perceptive Parkour Framework for Humanoids. The first target is simulation reproduction, not full real-robot deployment.

Core problem

Perceptive Humanoid Locomotion

Train a humanoid to move over rough terrain using depth perception and proprioception.

Key mechanism

Safer Footholds

Use terrain edge detection and foot safety points to reduce risky partial foot contacts.

Key mechanism

Flat Patch Commands

Sample feasible target patches so training does not reward fake shortcuts.

Learning Sequence

Level Focus Done When
1. Paper Reading Understand the problem, pipeline, observations, rewards, ablations, and failure modes. You can explain the paper without looking at it.
2. Codebase Reading Map Project Instinct task code, terrain code, rewards, observations, and policy flow. You know where each mechanism lives in the released code.
3. Isaac Lab Baseline Create or modify a simple locomotion task with rough terrain and velocity tracking. A baseline policy runs and logs rewards/reset reasons.
4. Perception Add depth or ray-cast observations, history, and observation randomization. You can compare blind locomotion with perception-conditioned locomotion.
5. Paper Mechanisms Add flat patch sampling, terrain edge detection, and simplified foot safety penalties. You can run ablations showing what each mechanism changes.
6. Mini Paper Write a report with method, simplifications, plots, screenshots, videos, and failure cases. The project is reproducible from clean instructions.

Six Week Plan

  1. Week 1: read the paper, install Isaac Lab, run one example, clone Project Instinct.
  2. Week 2: create a baseline locomotion task and record reward/reset statistics.
  3. Week 3: add stairs, gaps, ramps, platforms, and depth observation.
  4. Week 4: implement flat patch sampling and position-based commands.
  5. Week 5: implement terrain edge detection and simplified foot safety penalty.
  6. Week 6: run ablations, record videos, and write a mini paper-style report.

Senior-Level Checklist

  • Custom Isaac Lab task
  • Custom terrain generation
  • Custom observations and commands
  • Custom reward and termination terms
  • Depth/perception visualization
  • Ablation table and failure analysis
  • Clear reproducible README
