Learning Roadmap

Phase Goal Practice Output Status
1. Setup Install and validate Isaac Sim, GPU rendering, Python environment, and example scenes. Run a sample scene and capture a screenshot or short video. Not started
2. Core Isaac Sim Learn stage/world creation, assets, physics, cameras, sensors, and action graphs. Build a small mobile-robot scene with camera and lidar-like sensor output. Not started
3. ROS 2 Bridge Connect Isaac Sim to ROS 2 topics, TF, sensor messages, and robot control commands. Drive a simulated robot from ROS 2 and record topic output. Not started
4. Course Replication Reproduce selected Miami CSC398 or CSC752 Isaac Sim homework workflows. Complete laser scan, localization, or RRT-style practice in Isaac Sim. Not started
5. Robot Learning Bridge Cornell-style robot learning concepts from MuJoCo/Gymnasium into Isaac Sim or Isaac Lab. Implement a small reaching, navigation, or manipulation learning experiment. Not started
6. Portfolio Project Create one end-to-end demo combining simulation, sensors, control, and learning. Publish a README, video, and reproducible scripts. Not started
