# Isaac Sim Coursework and Learning Plan

This repository tracks Isaac Sim coursework references and a personal learning plan for building practical robotics simulation skills.

## Local Files

- `docs/index.html`: GitHub Pages landing page with the plan and course references.
- `docs/learning-plan.md`: editable learning tracker.
- `docs/coursework-research.md`: source notes from the coursework search.

## Deployment

The site is designed to deploy with GitHub Pages from the `docs` folder on the `main` branch.
